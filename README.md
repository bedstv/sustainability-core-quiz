# 永續發展核心模組測驗

以 C011、C012、C013 三份課程講義為範圍的靜態測驗網站，可直接部署至 GitHub Pages。

## 功能

- 三科單獨練習與核心模組綜合模擬
- 180 題四選一題庫，每題附解析與講義頁碼
- 正式模式：45 分鐘倒數、到時自動交卷
- 練習模式：作答後立即顯示答案
- 題目及選項隨機排列
- 歷次成績、各科正確率與錯題特訓
- 響應式版面、深色模式與鍵盤操作

## 本機預覽

在本目錄執行：

```bash
python3 -m http.server 8000
```

然後開啟 `http://localhost:8000`。

## 部署至 GitHub Pages

1. 將本目錄內容提交至 GitHub repository。
2. 在 repository 的 **Settings → Pages** 選擇 **Deploy from a branch**。
3. 選擇網站所在分支與資料夾，儲存後等待 GitHub 完成部署。

網站不需建置程序、後端或資料庫。學習紀錄只儲存在使用者瀏覽器的 `localStorage`。

## 題庫資料

題庫位於 `data/`，每題包含：

- 模組與子主題
- 難度
- 題目與四個選項
- 正確答案
- 簡短解析
- 講義頁碼

本網站不收錄或散布講義全文與投影片圖片。
