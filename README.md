# 永續發展核心模組測驗

以 C011、C012、C013、C021、C022、C023 六份課程講義為範圍的靜態測驗網站，可直接部署至 GitHub Pages。

## 功能

- 六份講義可單選或任意複選，自由組合模擬範圍
- 600 題四選一題庫，每份講義 100 題，每題附解析與講義頁碼
- 可切換智慧輪替與完全隨機抽題；智慧輪替優先抽取未出現、出現次數較少及最久未出現的題目
- 正式模式：45 分鐘倒數、到時自動交卷
- 練習模式：作答後立即顯示答案
- 各講義平均配題，題目及選項隨機排列
- 歷次成績、各科正確率與錯題特訓
- 響應式版面、深色模式與鍵盤操作
- 顯示題庫版號與線上驗證狀態，自動檢查新版並以版本參數更新快取

## 本機預覽

在本目錄執行：

```bash
python3 -m http.server 8000
```

然後開啟 `http://localhost:8000`。

## 部署至 GitHub Pages

1. 將本目錄內容提交至 GitHub repository。
2. 在 repository 的 **Settings → Pages** 選擇 **Deploy from a branch**。
3. 選擇網站所在分支與資料夾，儲存後等待 GitHub 完成部署。

網站不需建置程序、後端或資料庫。學習紀錄只儲存在使用者瀏覽器的 `localStorage`。

## 題庫資料

題庫位於 `data/`，每題包含：

- 模組與子主題
- 難度
- 題目與四個選項
- 正確答案
- 簡短解析
- 講義頁碼

本網站不收錄或散布講義全文與投影片圖片。
